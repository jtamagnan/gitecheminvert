# About

This is where all icons and flags are saved

# Flags

Every flag will be loaded via its language code for example if the
language code is `fr` then `fr_flag.pong` will be loaded.

## Flag size

Every flag is to be:
width: 512px
height: 352px

## Flag colors

red: #ee5281
blue: #288eea
yellow: #eabe41
white: #ffffff
